Flash Cards Stylées - Réduction des Endomorphismes et des Matrices Carrées

Quel est l'objectif principal de la réduction d'un endomorphisme ?
Rechercher une base où la matrice de l'endomorphisme est la plus simple possible (diagonale ou triangulaire) pour un espace vectoriel de dimension finie.
Définition d'une valeur propre `\(\lambda\)` d'un endomorphisme `\(u\)`.
`\(\lambda \in K\)` tel qu'il existe `\(x \in E, x \neq 0\)` avec `\(u(x) = \lambda x\)`.
Définition d'un vecteur propre.
Vecteur non nul `\(x \in E\)` colinéaire à `\(u(x)\)`, associé à une unique valeur propre `\(\lambda\)` (i.e., `\(u(x) = \lambda x\)`).
Qu'est-ce que le sous-espace propre `\(E_\lambda\)` ?
`\(E_\lambda = \Ker(u - \lambda \Id) = \{x \in E \mid u(x) = \lambda x\}\)`, stable par `\(u\)`.
Propriété des sous-espaces propres pour des valeurs propres distinctes.
La somme des `\(E_{\lambda_i}\)` (`\(\lambda_i\)` distinctes) est directe : `\(\bigoplus E_{\lambda_i}\)`, et une famille de vecteurs propres associés est libre.
Définition du polynôme caractéristique `\(\chi_u(\lambda)\)`.
`\(\chi_u(\lambda) = \det(\lambda \Id - A)\)`, de degré `\(n\)`, indépendant de la base. Valeurs propres = racines de `\(\chi_u\)`.
Multiplicité algébrique `\(m_a(\lambda)\)` et géométrique `\(m_g(\lambda)\)`.
`\(m_a(\lambda)\)` = ordre de `\(\lambda\)` comme racine de `\(\chi_u\)` ; `\(m_g(\lambda) = \dim E_\lambda\)`, avec `\(1 \leq m_g(\lambda) \leq m_a(\lambda)\)`.
Qu'est-ce qu'un polynôme scindé ?
Polynôme factorisable en produits de facteurs du premier degré (sur `\(\mathbb{C}\)`, tout polynôme est scindé par théorème de d'Alembert).
Conditions nécessaires et suffisantes (CNS) pour qu'un endomorphisme `\(u\)` soit diagonalisable.
Équivalentes à : base de vecteurs propres ; `\(E = \bigoplus E_\lambda\)` ; `\(\sum \dim E_\lambda = n\)` ; ou `\(\chi_u\)` scindé et `\(m_a(\lambda) = m_g(\lambda)\)` pour chaque `\(\lambda\)`.
Condition suffisante pour la diagonalisabilité.
`\(n\)` valeurs propres distinctes (racines simples de `\(\chi_u\)`) implique diagonalisable.
CNS pour qu'un endomorphisme `\(u\)` soit trigonalisable.
`\(\chi_u\)` est scindé. Sur `\(\mathbb{C}\)`, tout endomorphisme est trigonalisable.
Propriété de la matrice triangulaire dans la trigonalisation.
Diagonale = valeurs propres répétées selon leur multiplicité algébrique.
Application : Calcul de puissances d'une matrice diagonalisable `\(A = P D P^{-1}\)`.
`\(A^n = P D^n P^{-1}\)`, où `\(D^n\)` est facile à calculer (éléments `\(\lambda_i^n\)`).
Application : Puissances d'une matrice trigonalisable non diagonalisable.
Utiliser `\(A = P T P^{-1}\)` avec `\(T\)` triangulaire ; pour nilpotente (`\(\lambda = 0\)`), `\(N^k = 0\)` pour `\(k\)` suffisant.
Application aux suites récurrentes linéaires (e.g., ordre 2 : `\(u_{n+2} = a u_{n+1} + b u_n\)`).
Forme matricielle `\(U_{n+1} = A U_n\)` ; résoudre via diagonalisation (`\(u_n = c_1 \lambda_1^n + c_2 \lambda_2^n\)`) ou trigonalisation.
Application aux systèmes différentiels `\(X' = A X\)`.
Solutions forment un ev de dim `\(n\)` ; diagonalisable : `\(X(t) = P \diag(e^{\lambda_i t}) P^{-1} X(0)\)` ; sinon, résolution en cascade via trigonalisation.
Théorème de Cauchy-Lipschitz pour `\(X' = A X\)`.
Existence et unicité d'une solution avec condition initiale `\(X(0) = X_0\)`.
Exemple de valeurs propres pour un projecteur `\(p\)`.
Spectre `\(\{0, 1\}\)` ; `\(E_0 = \Ker p\)`, `\(E_1 = \Im p\)`.
Exemple de valeurs propres pour une symétrie vectorielle `\(s\)`.
Spectre `\(\{-1, 1\}\)` ; `\(E_{-1} = \Ker(s + \Id)\)`, `\(E_1 = \Ker(s - \Id)\)`.
Remarque sur le spectre et la trace.
Somme des valeurs propres (avec multiplicité) = trace de la matrice.
